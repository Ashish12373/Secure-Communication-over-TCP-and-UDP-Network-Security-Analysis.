# Secure-Communication-over-TCP-and-UDP-Network-Security-Analysis.
his project implements and analyzes secure and insecure transport-layer communication using UDP, TCP, TLS, and DTLS. It demonstrates IP spoofing, replay attacks, encrypted handshakes, DTLS anti-replay windows, and ICMP tunneling, supported by Python clients, OpenSSL servers, packet captures, and Wireshark traces.
